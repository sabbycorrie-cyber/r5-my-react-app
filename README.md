Jokes for Dayz
A React project using the Official Joke API to generate random jokes.

Features:
Fetch data from an external API
Displays joke setup and punchline
Interactive "Surprise Me!" button
Responsive design
Custom CSS styling, hover effects, and transitions

Built With:
React
JavaScript
CSS
Vite

API Used:
official_joke_api
https://official-joke-api.appspot.com/random_joke
